Aula voltado para a pratica

Realizamos uma aula onde foi nos enssinado quais tags usar e para que serve

Fieldset= Abre uma caixa onde será realizado a criação do forme

Legend= é usado para nomear o fieldset

Form= criação do nosso formulário, organizando os campos que fazem parte do mesmo envio de informações.

acttion= colocado dentro da tag legend dentro das setas e colocar o link de direcionamento

Label= mostra ao usuario o que deve ser preenchido em cada campo

Input= Abrir uma caixinha para digitar o que nomeamos em label

date= Permite escolher uma data pelo navegador
